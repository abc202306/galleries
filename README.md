1. requirements
	1. Obsidian (software)
	1. Git (Software)
	1. Git LFS (Git Extension)

```powershell
mkdir "./galleries/"
cd "./galleries/"
git clone "https://github.com/abc202306/galleries/"
git lfs pull
```
